# Irina Goremykina

> Personal Web Site

## Requirements

- [Node.js](https://nodejs.org/en/) 8.x
- [Yarn](https://yarnpkg.com/en/docs/install) package manager


## Quick Start

```sh
$ yarn
$ yarn run dev
$ open http://0.0.0.0:3000
```

## Build

```sh
$ yarn run build
$ yarn start
$ open http://0.0.0.0:3000
```
